# 🚀 Supply Chain Analytics Dashboard (Power BI)

An end-to-end **Supply Chain Delivery Performance Dashboard** built using **Power BI** to analyze delivery efficiency, identify root causes of failures, and provide actionable business insights.

---

## 📊 Problem Statement
Delivery performance was inconsistent, leading to poor customer service and operational inefficiencies.  
The goal was to analyze order data and answer:
- Why are deliveries late or incomplete?
- Where are the biggest performance gaps?
- What actions can improve delivery performance?

---

## 🎯 Project Objectives
- Track delivery KPIs (On-Time, In-Full, OTIF)
- Identify root causes of delivery failures
- Compare actual performance vs targets
- Provide clear, actionable recommendations

---

## 📈 Key Insights
- Only **29% of orders were delivered On-Time In-Full (OTIF)**
- Identified a **-37% performance gap vs target**
- Delivery issues are **system-wide (not city-specific)**
- Key problems found in:
  - Inventory planning  
  - Delivery coordination  
  - High-risk customers

## 📊 Dashboard Features

- Product Category Analysis
- City-wise Performance

### 🔹 Page 2: Root Cause & Gap Analysis
- Late Delivery % by Customer, City, Category
- KPI vs Target Comparison
- Performance Gap Analysis (-37%)
- Actionable Recommendation Section

### 🔹 Page 3: Executive Summary
- High-level KPIs
- Key insights for decision-makers
- Business-focused summary

---

## 📌 Key Metrics Explained
- **On-Time Delivery %** → Orders delivered on time  
- **In-Full Delivery %** → Orders delivered completely  
- **OTIF %** → Orders delivered both on time and in full  

---

## 💡 Recommendations
- Improve inventory planning to ensure product availability  
- Strengthen coordination between warehouse and delivery teams  
- Focus on high-risk customers and categories  
- Optimize delivery scheduling and logistics
